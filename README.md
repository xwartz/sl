# SL

A trading position calculator for crypto and stock trading. Calculate position size, leverage, liquidation price, and risk/reward ratio to assist your trading plan.

## Features

**Multiple Plans**: Create and manage multiple trading plans with tabs

**Real-time Calculation**:
- Position size calculation
- Average entry price
- Liquidation price
- Actual leverage
- Total fees
- Risk/Reward ratio

**Order Management**:
- Add multiple entry orders
- Set unit price and order volume for each entry
- Duplicate or delete orders easily
- View cumulative investment and shares

**Direction Support**:
- Long positions
- Short positions

**Configurable**:
- Leverage (1x - 125x)
- Fee rate
- Current market price

**Data Persistence**: All your plans and orders are saved locally in your browser

**i18n Support**: English and Chinese (简体中文)

**Theme**: Light and dark mode support

## Usage

1. **Configure Your Position**:
   - Select direction (Long/Short)
   - Set leverage multiplier
   - Enter fee rate (%)
   - Input current market price

2. **Plan Your Orders**:
   - Add multiple entry orders
   - Set unit price and order volume for each order
   - View real-time calculations for each order

3. **Review Summary**:
   - Total investment
   - Contract value
   - Average entry price
   - Liquidation price
   - Actual leverage
   - Total fees

4. **Manage Plans**:
   - Create multiple trading plans using tabs
   - Double-click tab names to rename
   - Switch between different plans easily
