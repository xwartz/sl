# 交易仓位计算器

计算仓位大小、杠杆倍数、爆仓价格和盈亏比，辅助交易计划

## 功能特点

**多计划管理**：使用标签页创建和管理多个交易计划

**实时计算**：
- 仓位大小计算
- 平均入场价
- 爆仓价格
- 实际杠杆倍数
- 总手续费
- 盈亏比

**订单管理**：
- 添加多笔入场订单
- 为每笔订单设置挂单价格和投入金额
- 轻松复制或删除订单
- 查看累计投资和持仓数量

**方向支持**：
- 做多仓位
- 做空仓位

**可配置**：
- 杠杆倍数（1x - 125x）
- 费率
- 当前市场价格

**数据持久化**：所有计划和订单都保存在浏览器本地存储中

**国际化**：支持英文和中文

**主题**：支持亮色和暗色模式

## 使用方法

1. **配置您的仓位**：
   - 选择方向（做多/做空）
   - 设置杠杆倍数
   - 输入费率（%）
   - 输入当前市场价格

2. **规划您的订单**：
   - 添加多笔入场订单
   - 为每笔订单设置挂单价格和投入金额
   - 实时查看每笔订单的计算结果

3. **查看汇总**：
   - 总投入本金
   - 合约总价值
   - 平均入场价
   - 爆仓价格
   - 实际杠杆倍数
   - 总手续费

4. **管理计划**：
   - 使用标签页创建多个交易计划
   - 双击标签名称重命名
   - 轻松切换不同计划
